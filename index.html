Bienvenido a Javascript Latino

"Compartir conocimiento es la forma más poética de luchar contra la opresión"

Desarrollo

La comunidad tiene la intención de desarrollar productos open sources que sean de utilidad para todos, contamos con un repositorio en github donde se podrán descargar.

Conocimiento

Nuestra filosofía es el conocimiento libre, por lo cual estamos en la onda de aprender y compartir conocimiento no importa tu nivel de conocimiento, seguramente lo que aportes será de utilidad para alguien mas ;)

Open Source

Nuestro objetivo es realizar todo bajo licencia Open Source lo cual cualquiera que necesite o este interesado puede tener acceso a nuestro material.
